タグ一覧

playamov

属性必須?値説明
storage yes ファイル名 再生するファイル名を指定します。
slot no ムービーオブジェクト番号(0〜) この操作を行うムービーオブジェクト番号を指定します。
省略すると 0 が指定されたとみなされます。
left no 左位置 動画の左位置を指定します。
top no 上位置 動画の上位置を指定します。
loop no true/false ループ再生するかどうかを指定します。
省略すると false が指定されたとみなされます。

ビデオを再生します。


stopamov

属性必須?値説明
slot no ムービーオブジェクト番号(0〜) この操作を行うムービーオブジェクト番号を指定します。
省略すると 0 が指定されたとみなされます。

ビデオを停止します。


nextamov

属性必須?値説明
storage yes ファイル名 次に再生するファイルを指定します。
slot no ムービーオブジェクト番号(0〜) この操作を行うムービーオブジェクト番号を指定します。
省略すると 0 が指定されたとみなされます。
loop no true/false 次に再生するファイルをループ再生するかどうかを指定します。
省略すると false が指定されたとみなされます。

指定したファイルを現在の動画再生が終了した後、シームレスに再生継続します。
現在の動画がループ中の場合は次の動画には移りません。


amovopt

属性必須?値説明
slot no ムービーオブジェクト番号(0〜) この操作を行うムービーオブジェクト番号を指定します。
省略すると 0 が指定されたとみなされます。
left no 左位置 動画の左位置を指定します。
top no 上位置 動画の上位置を指定します。
loop no true/false ループ再生するかどうかを指定します。
省略すると false が指定されたとみなされます。
visible no true/false ビデオの表示/非表示を指定します。
frame no フレーム番号 指定されたフレームにシークします。
opacity no 0(完全透明) 〜 255(完全不透明) 不透明度を指定します。

ビデオの属性を設定します。


wam

属性必須?値説明
slot no ムービーオブジェクト番号(0〜) この操作を行うムービーオブジェクト番号を指定します。
省略すると 0 が指定されたとみなされます。
canskip no true/false クリックによるスキップを可能にするかどうかを指定します。
省略すると true が指定されたとみなされます。

再生中の動画が停止するのを待ちます。
動画が再生されていない場合は待ちません。


wamnext

属性必須?値説明
slot no ムービーオブジェクト番号(0〜) この操作を行うムービーオブジェクト番号を指定します。
省略すると 0 が指定されたとみなされます。
canskip no true/false クリックによるスキップを可能にするかどうかを指定します。
省略すると true が指定されたとみなされます。

次の動画へ切り替わるのを待ちます。
次に再生する動画が設定されていない時は待ちません。


amovmove

属性必須?値説明
slot no ムービーオブジェクト番号(0〜) この操作を行うムービーオブジェクト番号を指定します。
省略すると 0 が指定されたとみなされます。
fps no フレームレート path で指定されたフレームをどの程度の速度で進めるかを FPS で指定します。
省略すると 30 が指定されたとみなされます。
path yes フレームと opacity 値を組み合わせた文字列 どのように opacity 値を変化させるかを指定します。
"(15,255)"のように指定すると、15フレームで opacity 値を現在の値から 255 にします。

opacity 値を連続的に変化させてフェードイン・フェードアウトのような効果を作り出します。
wm タグを使って終了を待つことが出来ます。
